面積計算ツール

正方形・長方形・三角形・円・台形・ひし形の面積を計算。坪・畳・アール・ヘクタールへの換算も自動表示します。

面積計算ツールの使い方

面積計算ツールは、さまざまな図形の面積を簡単に計算できる無料オンラインツールです。不動産の面積確認、土地の測量結果の換算、学校の数学の課題など、幅広い用途でお使いいただけます。

対応する図形と計算式

以下の6つの図形に対応しています。図形を選択すると、必要な入力項目が自動的に表示されます。

  • 正方形：面積 = 一辺 x 一辺
  • 長方形：面積 = 縦 x 横
  • 三角形：面積 = 底辺 x 高さ / 2
  • 円：面積 = 円周率 x 半径 x 半径
  • 台形：面積 = (上底 + 下底) x 高さ / 2
  • ひし形：面積 = 対角線1 x 対角線2 / 2

単位換算について

計算結果は平方メートル（m2）のほか、以下の単位に自動変換して表示します。

  • 坪：1坪 = 約3.30579 m2。日本の不動産取引で広く使われる単位です。
  • 畳（江戸間）：1畳 = 約1.548 m2。部屋の広さを表すのに使われます。
  • a（アール）：1a = 100 m2。農地などの面積に使われます。
  • ha（ヘクタール）：1ha = 10,000 m2。広大な土地の面積に使われます。

よくある質問（FAQ）

Q. 坪と平方メートルの換算方法を教えてください。

A. 1坪は約3.30579平方メートルです。平方メートルから坪に変換するには、平方メートルの値を3.30579で割ります。逆に、坪から平方メートルに変換するには3.30579を掛けます。例えば、30坪は約99.17平方メートルです。

Q. 畳の大きさは地域によって異なりますか？

A. はい、畳にはいくつかの規格があります。本ツールでは江戸間（1畳 = 約1.548 m2、176cm x 88cm）を基準にしています。京間（191cm x 95.5cm）は約1.824 m2、中京間（182cm x 91cm）は約1.656 m2です。お住まいの地域の畳の規格に応じて読み替えてください。

Q. 円の面積計算で使われる円周率は？

A. 本ツールではJavaScriptのMath.PI（3.141592653589793...）を使用して計算しています。十分な精度で面積を算出できます。

Q. 不動産の面積確認に使えますか？

A. 概算値としてお使いいただけます。ただし、実際の土地や建物は複雑な形状をしていることが多いため、正確な面積は登記簿謄本や測量図でご確認ください。本ツールは長方形や三角形などの単純な図形の面積を素早く計算するのに適しています。
