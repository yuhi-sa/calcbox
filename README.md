# CalcBox 🧮

61種類以上の暮らしに役立つ無料オンライン計算ツール集。BMI計算、消費税計算、ローン計算、カロリー計算、給与手取り計算など、日常のちょっとした計算をサッと解決します。

🌐 **[https://yuhi-sa.github.io/calcbox/](https://yuhi-sa.github.io/calcbox/)** | [English](https://yuhi-sa.github.io/calcbox/en/)

## ✨ 特徴

- 🆓 **完全無料・登録不要** - ブラウザだけで即利用
- 🔒 **プライバシー重視** - データはすべてブラウザ内で処理、サーバーへの送信なし
- 📱 **PWA対応** - スマホのホーム画面に追加してアプリ感覚で利用可能
- 🌏 **多言語対応** - 日本語 / English
- 🌙 **ダークモード** - ライト/ダークテーマ切替対応
- ⭐ **お気に入り機能** - よく使うツールをブックマーク

## 🚀 ローカル開発

```bash
git clone https://github.com/yuhi-sa/calcbox.git
cd calcbox
# 任意のHTTPサーバーで起動
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。

## 🛠️ 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JavaScript（フレームワーク不使用）
- **ホスティング**: GitHub Pages
- **データ処理**: すべてクライアントサイドで完結（サーバー不要）
- **PWA**: Service Worker によるオフライン対応
