How to Use the IP Address / Subnet Calculator

Enter an IP address, select a CIDR prefix length, and click "Calculate" to display the network information.

What is CIDR?

CIDR (Classless Inter-Domain Routing) is a notation that specifies the number of bits in the network portion of an IP address. For example, /24 means the upper 24 bits are the network portion.

Calculated Information

  • Subnet Mask: The mask value indicating the network portion
  • Network Address: The first address of the subnet
  • Broadcast Address: The last address of the subnet
  • Host Range: The range of usable host addresses
  • Available Hosts: The number of addresses assignable to hosts
